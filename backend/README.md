# 实时同声传译系统 - 后端服务文档

## 📖 项目概述

实时同声传译系统后端服务，基于 FastAPI + WebSocket 实现低延迟流式音频处理，集成第三方 ASR、翻译和 TTS 服务，提供完整的语音识别→文本翻译→语音合成闭环能力。

### 核心特性

- ✅ **流式处理架构**：实时接收音频流，边说边译，延迟 < 2秒
- ✅ **多厂商支持**：集成 Azure、讯飞、DeepL 等专业服务
- ✅ **智能缓存优化**：LRU 缓存高频短语，降低 API 调用成本
- ✅ **模块化设计**：ASR、翻译、TTS 独立服务，易于替换和扩展
- ✅ **高可用性**：完善的异常处理和日志记录
