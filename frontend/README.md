# 实时同声传译系统 - 前端文档

## 📖 系统概述

实时同声传译系统是一款基于 React 18+ 和 Python FastAPI 的专业级语音翻译应用，提供从语音输入到翻译输出的完整闭环流程。系统支持多语言实时翻译，采用流式处理架构，延迟低于2秒，适用于会议、教学、跨国交流等场景。

### 核心特性

- ✅ **实时流式处理**：边说边译，延迟 < 2秒
- ✅ **多语言支持**：中文、英语、日语、韩语、法语、德语、西班牙语、俄语
- ✅ **专业级翻译**：集成 DeepL、Azure 等专业服务
- ✅ **音频优化**：OPUS 压缩编码，节省 90% 带宽
- ✅ **智能静音检测**：VAD 技术过滤无效音频
- ✅ **可视化展示**：实时音频波形、文本滚动效果
- ✅ **响应式设计**：支持移动端和桌面端
